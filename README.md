📊 Fraud Detection Dashboard (Power BI)

This project presents an interactive Fraud Detection Dashboard built using Power BI, designed to analyze transaction data and uncover patterns related to fraudulent activity across different dimensions.

🔍 Overview

The dashboard provides a comprehensive view of transaction behavior, risk scores, and fraud distribution across merchant categories, transaction types, users, and geographic regions. It enables quick identification of anomalies and high-risk segments through intuitive visualizations.

🚀 Key Features
Fraud Distribution by Merchant Category
Visual breakdown of fraud occurrences across categories like Clothing, Electronics, Grocery, Travel, and Food.
Risk Score Analysis
Comparison of transaction amounts alongside associated IP risk scores to highlight suspicious activity.
Transaction Trends by Type
Analysis of transaction volume across POS, ATM, QR, and Online channels.
User-Level Insights
High-volume transaction activity visualization by user IDs to detect unusual patterns.
Geographic Insights
Country-level transaction distribution to identify regional fraud trends.
KPI Metrics
Total Transaction Amount: 1.78M
Total Transactions: 142K
Interactive Filters
Dynamic filtering by transaction amount and other parameters for deeper analysis.
